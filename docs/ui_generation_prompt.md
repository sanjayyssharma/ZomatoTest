# UI Generation Prompt for AI Recommender

You can copy and paste the following prompt into your UI generation tool (e.g., Google Snitch or any design AI) to generate mockups and frontend code for your Next.js application.

---

**Prompt:**

```text
Create a modern, premium UI design for a "Full-Stack AI Restaurant Recommender" web application built with Next.js and Tailwind CSS. The design should feature a sleek, dark-mode aesthetic with glassmorphism elements, vibrant gradient accents, and micro-animations to make it feel highly dynamic.

The application has the following key views and components:

1. Hero / Search Section:
   - A bold, typography-driven header (e.g., "Discover Your Next Favorite Meal").
   - A responsive search form with inputs for:
     - Location (Text input or dropdown)
     - Budget (Options: Low, Medium, High)
     - Minimum Rating (Slider or star selection)
     - Cuisines (Multi-select tags)
   - A prominent, glowing "Get Recommendations" primary button with a subtle hover effect.

2. Results / Recommendations List:
   - A skeleton loading state while the AI is fetching results.
   - A grid or list of "Restaurant Cards" for the recommended options.
   - Each card should display:
     - Restaurant name, rating (with a star icon), and cuisine tags.
     - Cost for two and location.
     - An "AI Explanation" section styled distinctly (e.g., a frosted glass container with a subtle glowing border) showing why the LLM recommended this specific place based on the user's filters.
   - An empty state illustration or message if no restaurants match the criteria.

The overall layout should be fully responsive (mobile, tablet, desktop). Focus on high-quality UX/UI practices, utilizing a curated color palette (deep blacks/grays for background, vibrant neon accents for highlights, and stark white for primary text) and modern Google Fonts (like Inter or Outfit). Ensure the interface looks professional, production-ready, and aesthetically "wow."
```
